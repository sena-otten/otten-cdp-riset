SOP & Operations

Standard Operating Procedures dan Checklist untuk operasional CDP

5.1 Sales SOP (B2B)
Scope: SOP untuk tim Sales B2B dalam menggunakan CRM, mengelola pipeline, dan melakukan visit.

📋 SOP-SALES-001: Lead Management

📅 Frequency: Daily 👤 Owner: Sales Rep ✅ Approval: Sales Manager
  1. Lead Entry: Input lead baru ke CRM dalam waktu 1x24 jam setelah diterima
  2. Data Validation: Pastikan mandatory fields terisi (Company Name, Contact Person, Phone/WA)
  3. Initial Contact: Lakukan first contact dalam 2x24 jam
  4. Lead Qualification: Update status dan scoring berdasarkan BANT criteria
  5. Follow-up Schedule: Set reminder untuk follow-up berikutnya
  6. Conversion/Disqualify: Convert ke Deal atau mark sebagai Disqualified dengan alasan

📍 SOP-SALES-002: Visit Planning & Execution

📅 Frequency: Weekly 👤 Owner: Sales Rep ✅ Approval: Sales Manager
  1. Weekly Planning (Jumat): Submit visit plan untuk minggu depan via CRM
  2. Route Optimization: Gunakan fitur route planning untuk efisiensi
  3. Pre-Visit Prep: Review customer profile dan history sebelum visit
  4. Check-in: Lakukan GPS check-in saat tiba di lokasi
  5. Visit Report: Input visit report dengan foto dan notes dalam 2 jam setelah visit
  6. Follow-up Action: Create task/reminder untuk action items dari visit

💰 SOP-SALES-003: Deal & Pipeline Management

📅 Frequency: Daily 👤 Owner: Sales Rep ✅ Approval: Sales Manager
  1. Deal Creation: Create deal setelah lead qualified dengan estimasi value
  2. Stage Update: Update pipeline stage setiap ada progress (max 3 hari)
  3. Probability: Adjust win probability berdasarkan kondisi aktual
  4. Quotation: Generate quotation via system untuk tracking
  5. Negotiation Notes: Log semua negotiation points dan competitor info
  6. Won/Lost: Update status final dengan detail alasan

Quick Reference: Sales Metrics Target

Metric Target Frequency

Customer Visits Min 15 visits/minggu Weekly
Visit Report Completion 100% dalam 2 jam Daily
Pipeline Update Max 3 hari stale Daily
Conversion Rate Min 20% lead to deal Monthly
5.2 Customer Service SOP
Scope: SOP untuk tim Customer Service dalam menangani inquiry, complaint, dan support request.

🎫 SOP-CS-001: Ticket Handling

📅 Frequency: Per Ticket 👤 Owner: CS Agent ✅ SLA: Based on Priority
  1. Ticket Receipt: Acknowledge ticket dalam 15 menit (auto-reply)
  2. Categorization: Classify ticket (Inquiry/Complaint/Request/Technical)
  3. Priority Assignment: Set priority berdasarkan urgency dan impact
  4. First Response: Berikan response pertama sesuai SLA priority
  5. Resolution: Handle atau escalate sesuai kategori
  6. Closure: Confirm resolution dengan customer dan close ticket
  7. CSAT Survey: Trigger survey setelah ticket closed

🚨 SOP-CS-002: Escalation Matrix

📅 Frequency: As Needed 👤 Owner: CS Agent → CS Lead → CS Manager
Level Trigger Escalate To Response Time
L1 General inquiry, FAQ CS Agent 4 jam
L2 Complex issue, SLA breach CS Lead 2 jam
L3 Major complaint, VIP customer CS Manager 1 jam
Executive Legal threat, media exposure Head of Operations 30 menit

5.3 After Sales SOP
Scope: SOP untuk tim After Sales dalam menangani service mesin, warranty, dan spare parts.

🔧 SOP-AS-001: Service Request Handling

📅 Frequency: Per Request 👤 Owner: Service Coordinator ✅ SLA: Based on Priority
  1. Request Receipt: Log service request ke system dengan detail masalah
  2. Warranty Check: Verifikasi status warranty mesin
  3. Priority Assignment: Tentukan priority (Critical/High/Medium/Low)
  4. Technician Assignment: Assign teknisi berdasarkan skill dan availability
  5. Schedule Confirmation: Konfirmasi jadwal service ke customer
  6. Service Execution: Teknisi execute service dan update status real-time
  7. Completion & Sign-off: Customer sign-off dan feedback collection

📅 SOP-AS-002: Preventive Maintenance Schedule

📅 Frequency: Monthly 👤 Owner: Service Coordinator
  1. Generate Schedule: System auto-generate PM schedule berdasarkan machine data
  2. Customer Notification: Kirim reminder H-7 dan H-1 sebelum jadwal PM
  3. Confirmation: Confirm atau reschedule dengan customer
  4. PM Execution: Teknisi lakukan PM checklist lengkap
  5. Report & Recommendation: Submit PM report dengan rekomendasi jika ada
  6. Next Schedule: System auto-set jadwal PM berikutnya

📦 SOP-AS-003: Spare Parts Management

📅 Frequency: Per Request 👤 Owner: Parts Coordinator
  1. Parts Request: Teknisi submit parts request via mobile app
  2. Stock Check: System auto-check stock availability
  3. Quotation: Generate quotation jika parts tidak dalam warranty
  4. Customer Approval: Get approval untuk parts berbayar
  5. Parts Issuance: Issue parts dari warehouse ke teknisi
  6. Installation: Teknisi install dan update serial number jika applicable
  7. Stock Update: System auto-update inventory

Service Priority Matrix

Priority Condition Response Time Resolution Target
Critical Mesin tidak bisa operasi sama sekali 2 jam 24 jam
High Mesin beroperasi tapi ada masalah signifikan 4 jam 48 jam
Medium Minor issue, tidak mengganggu operasi 24 jam 72 jam
Low Preventive maintenance, request info 48 jam 1 minggu
5.4 Marketing SOP
Scope: SOP untuk tim Marketing dalam menjalankan campaign, automation, dan content management.

📢 SOP-MKT-001: Campaign Creation & Execution

📅 Frequency: Per Campaign 👤 Owner: Marketing Executive ✅ Approval: Marketing Manager
  1. Campaign Brief: Create campaign brief dengan objective, target, dan budget
  2. Audience Selection: Define segment target menggunakan CDP segmentation
  3. Content Creation: Prepare creative assets dan copy
  4. Channel Setup: Setup campaign di masing-masing channel (Email, Push, WA, Ads)
  5. Testing: A/B test untuk subject line, creative, atau timing
  6. Approval: Submit untuk approval sebelum launch
  7. Launch & Monitor: Launch campaign dan monitor real-time metrics
  8. Post-Campaign Report: Generate report dalam 7 hari setelah campaign end

⚙ SOP-MKT-002: Automation Workflow Setup

📅 Frequency: Per Workflow 👤 Owner: Marketing Automation Specialist ✅ Approval: Marketing Manager
  1. Workflow Design: Define trigger, conditions, dan actions
  2. Content Preparation: Prepare semua content untuk setiap step
  3. Build in System: Setup workflow di marketing automation platform
  4. Test Run: Test dengan internal email/account
  5. QA Review: Review oleh team member lain
  6. Soft Launch: Launch ke small segment (5-10%) untuk validation
  7. Full Launch: Activate untuk full audience setelah validation OK
  8. Monitoring: Set up alerts untuk anomaly detection

🎁 SOP-MKT-003: Voucher & Promo Management

📅 Frequency: Per Promo 👤 Owner: Promo Manager ✅ Approval: Marketing Manager + Finance
  1. Promo Planning: Define promo mechanics, budget, dan target ROI
  2. Budget Approval: Get approval dari Finance untuk budget allocation
  3. Voucher Setup: Create voucher codes dengan rules dan limitations
  4. Testing: Test voucher application di staging environment
  5. Distribution: Distribute melalui channel yang ditentukan
  6. Monitoring: Monitor redemption rate dan fraud detection
  7. Reconciliation: Reconcile dengan Finance di akhir periode

Campaign Approval Matrix

Budget Range Approver Lead Time

Rp 5-25 juta Marketing Manager + Finance 3 hari
Rp 25-100 juta Head of Marketing + CFO 5 hari
> Rp 100 juta CEO + Board 2 minggu
